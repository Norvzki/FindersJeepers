This project is best run using **Visual Studio Community**. You *can* use other tools if you know how to manually build and run .NET projects, but Visual Studio is recommended for convenience.

---

## Please see the guide below to get started.

### First, Clone the Repository

```bash
git clone git@github.com:SpinellyA/FindersJeepers.git
cd FindersJeepers
```

---

### Then, Restore Dependencies

This project uses packages like Entity Framework Core. Restore them using:

```bash
dotnet restore
```

---

## Database Setup

This project uses **PostgreSQL via Npgsql**.

Database setup instructions will be added soon.

---

## Running the Project

Open the sln file (not the csproj file)
and press run :)

---

## Notes

* Make sure you have the .NET SDK installed.
* Ensure PostgreSQL is running before starting the app.
* Update your connection string if needed.
